Политика конфиденциальности

Настоящая политика конфиденциальности применяется к мобильному приложению TubusMobile (далее — «Приложение»), созданному Алексеем Симоновым (далее — «Поставщик услуг») в качестве бесплатного сервиса. Приложение предназначено для использования «КАК ЕСТЬ».


Сбор и использование информации

Приложение собирает информацию при загрузке и использовании. Эта информация может включать:

  • IP-адрес вашего устройства
  • Просматриваемые страницы Приложения, время и дата посещения, время, проведенное на этих страницах
  • Общее время использования Приложения
  • Операционная система вашего мобильного устройства


Приложение не собирает точные данные о местоположении вашего устройства.

Приложение собирает данные о местоположении вашего устройства, что помогает Поставщику услуг определить приблизительное географическое положение и использовать их следующим образом:

  • Геолокационные сервисы: Поставщик услуг использует данные о местоположении для предоставления персонализированного контента, релевантных рекомендаций и сервисов на основе местоположения.
  • Аналитика и улучшения: Агрегированные и анонимизированные данные о местоположении помогают Поставщику услуг анализировать поведение пользователей, выявлять тенденции и улучшать функциональность Приложения.
  • Сторонние сервисы: Периодически Поставщик услуг может передавать анонимизированные данные о местоположении внешним сервисам для улучшения Приложения.

Поставщик услуг может использовать предоставленную вами информацию для связи с вами, отправки важных уведомлений и маркетинговых материалов.


Для улучшения работы Приложения Поставщик услуг может запросить определенную персональную информацию, которая будет храниться и использоваться в соответствии с данной политикой.


Доступ третьих лиц

Только агрегированные и анонимизированные данные периодически передаются внешним сервисам для улучшения Приложения. Поставщик услуг может делиться вашей информацией с третьими лицами, как описано в настоящей политике.


Поставщик услуг может раскрывать предоставленную пользователем и автоматически собранную информацию:

  • если это требуется по закону (например, по запросу суда);
  • для защиты прав, безопасности пользователей или расследования мошенничества;
  • доверенным поставщикам услуг, которые соблюдают условия данной политики.


Право на отказ

Вы можете прекратить сбор информации, просто удалив Приложение стандартным способом.


Политика хранения данных

Поставщик услуг хранит предоставленные пользователем данные, пока вы используете Приложение, и в течение разумного срока после. Для удаления данных напишите на wert001@yandex.ru.


Дети

Приложение не предназначено для сбора данных или маркетинга для детей младше 13 лет.


Поставщик услуг не собирает персональные данные детей. Родителям рекомендуется контролировать использование детьми Интернета. Если ребенок предоставил персональные данные, свяжитесь с Поставщиком услуг (wert001@yandex.ru). В некоторых странах для обработки персональных данных требуется согласие родителей (для лиц младше 16 лет).


Безопасность

Поставщик услуг принимает меры для защиты конфиденциальности вашей информации.


Изменения

Политика может обновляться. Поставщик услуг уведомит об изменениях, опубликовав новую версию на этой странице. Регулярно проверяйте политику — дальнейшее использование Приложения означает согласие с изменениями.

Ваше согласие

Используя Приложение, вы соглашаетесь с обработкой информации в соответствии с данной Политикой.


Контакты
